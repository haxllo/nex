# Nex Documentation

Nex is a keyboard-first launcher for Windows, built in Rust. It provides a global hotkey-activated floating search bar with fuzzy search across apps, files, folders, custom actions, and clipboard history.

For the latest plans and roadmap, see the [Plans](./plans/) directory.

## Key Features

- Global hotkey (`Ctrl+Space` by default)
- Fuzzy search with usage-based ranking
- Everything SDK integration for instant file search
- Clipboard history
- Plugin SDK (preview)
- WebView2 overlay with animations
