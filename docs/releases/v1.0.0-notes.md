# Nex v1.0.0 Release Notes

Release date: 2026-05-17

## Summary

First stable release! Nex is a keyboard-first launcher for Windows.

## Features

- Global Hotkey (Ctrl+Space by default)
- Fuzzy search for apps, files, folders
- Custom actions and web search
- Clipboard history (optional)
- Plugin support
- Game mode

## Installation

### Via Installer
Download `nex-1.0.0-windows-x64-setup.exe` and run.

### Via Portable
Download `nex-1.0.0-windows-x64.zip`, extract, and run `nex.exe`.

### Via Cargo
```bash
cargo install nex-cli
```

## Upgrading

Run the new installer or replace `nex.exe` with the new version.

## Known Issues

- Game mode detection may not work for all games
- Hotkey conflicts show warning but don't block startup
