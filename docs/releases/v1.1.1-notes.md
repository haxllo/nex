# Nex v1.1.1 Release Notes

**Release date:** 2026-05-24
**Channel:** stable

---

## Summary

Nex v1.1.1 is a performance release that moves search processing off the UI thread. The overlay no longer freezes during query execution, making the launcher feel noticeably more responsive during rapid typing and complex searches.

---

## What's Changed

### Async Search Engine

Search is now fully asynchronous. When you type, the query is dispatched to a background worker thread while the UI stays responsive. Results arrive via Windows message notification and are applied as soon as they're ready:

- **Before:** Every keystroke blocked the UI thread for the duration of the search (often 5–50ms, spiking higher for uncached queries).
- **After:** The UI thread never waits. Search runs in parallel, stale queries are automatically drained (only the latest keystroke matters), and results are posted back seamlessly.

### Under the Hood

- **`SearchWorker`** — Dedicated background thread with mpsc channels for request/result passing. Uses `PostMessageW` to notify the UI thread when results are ready.
- **Generation-based staleness** — Each request carries a generation counter. Results from superseded queries are silently dropped.
- **Thread-safe CoreService** — Wrapped in `Arc<Mutex<>>` to guarantee the SQLite connection is never accessed from multiple threads simultaneously.
- **Clean shutdown** — Worker thread exits gracefully when the runtime shuts down, with `Drop` joining the thread.

## Upgrade Notes

### From v1.1.0
Run the new setup.exe over your existing installation. Config and index database in `%APPDATA%\Nex` are preserved automatically.

### Fresh Install
Download and run `nex-1.1.1-windows-x64-setup.exe`. No prerequisites — everything is self-contained.

### Via Cargo
```bash
cargo install nex-cli --version 1.1.1
```

---

## Known Issues

Same as v1.1.0:
- **Calculator** supports basic arithmetic only.
- **Window Management** is command-palette-only (configurable hotkey bindings planned).
- **Plugin SDK** remains in preview.
- **Installer** is unsigned (SmartScreen warning expected).
