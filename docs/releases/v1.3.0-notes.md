# Nex v1.3.0 Release Notes

Release date: 2026-05-28

## Summary

This release completes the migration from the hybrid D2D/DWrite + GDI rendering stack to a unified GDI+-only pipeline, replaces the bundled SpaceMono font with Inter, and makes GDI+ a hard requirement. Also includes fixes for hanging CI tests, CI workflow corrections, and removal of stale documentation.

## Breaking Changes

### D2D/DWrite rendering stack removed

All rendering now goes through GDI+ exclusively:

- Panel background, help tip, footer, and list row highlights all use GDI+
- The `d2d_renderer.rs` module (~800 lines) has been deleted
- `GdipSetTextRenderingHint` with `TextRenderingHintClearTypeGridFit` is set on every graphics context
- GDI+ initialization failure is now fatal â€” the overlay posts `WM_CLOSE` instead of warning about a non-existent fallback

### SpaceMono replaced with Inter

Inter v4.1 (Regular, Bold, Medium, SemiBold) is now the bundled private font. When private fonts load successfully, the UI uses `"Inter"` instead of `"Segoe UI"`. The SpaceMono font directory has been removed.
