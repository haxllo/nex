# Nex v2.4.0 — Quick Launch

## ✨ New Features

### Quick Launch with Pinned Apps
- **Pin apps** to Quick Launch for instant access when overlay opens empty
- **Pin from search results** — click the bookmark icon to pin any app
- **Pinned apps at top** — search results prioritize pinned items
- **Instant feedback** — pin/unpin updates immediately without closing search
- **Input stays focused** — continue typing after pinning

### Improved Icon Quality
- **ExtractIconExW** — extracts all available icon sizes from executables
- Uses highest resolution available (32px, 48px, 256px depending on app)
- Falls back to SHGetFileInfo for shell URIs

## 🔧 Changes

- Quick Launch shows **only pinned items** when pins exist (no auto-fill)
- All pinned items shown regardless of max_items limit
- Auto-fill from usage only when no items are pinned
- Config uses nested TOML format for `quick_launch` section
- Pin icon shows filled state (★) for already-pinned apps

## 🐛 Fixes

- Fixed JavaScript syntax error that prevented overlay from opening
- Fixed config migration for old flat format keys
- Fixed path comparison normalization (handles `/` vs `\` and case)

## 📦 Files Changed

- `apps/core/assets/app.js` — Pin/unpin icons, Quick Launch rendering
- `apps/core/assets/style.css` — Pin icon styles
- `apps/core/src/config.rs` — QuickLaunchConfig, migrations
- `apps/core/src/index_store.rs` — Quick Launch queries, launch tracking
- `apps/core/src/overlay/icons.rs` — ExtractIconExW for better icons
- `apps/core/src/runtime_loop.rs` — Pin/unpin handlers, search sorting
