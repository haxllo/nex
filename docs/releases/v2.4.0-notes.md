Quick Launch with pinned apps and improved icon quality.

## Changelog (14 commits since v2.3.1)

### Performance

- **Icon extraction**: Uses ExtractIconExW to get highest resolution icons from executables (32px, 48px, or 256px)

### Bug Fixes

- **Overlay not opening**: Fixed JavaScript syntax error in isItemPinned regex that prevented WebView from signaling ready
- **Config format**: Fixed nested TOML structure for quick_launch settings, added migration for old flat format
- **Path normalization**: Fixed comparison handling `/` vs `\` and case differences

### Architecture

- **Quick Launch**: Hybrid model — shows only pinned items when pins exist, auto-fills from usage when no pins
- **Search sorting**: Pinned items appear at top of search results
- **Pin/unpin**: Updates overlay immediately without closing search, input retains focus
