WebView stays warm for instant re-open; icon cache cleared on idle to reclaim heap.

## Changelog (11 commits since v2.4.0)

### Performance

- **Warm WebView**: Overlay WebView now stays resident for process lifetime. Warm-release timer clears decoded icon cache only, keeping page loaded for consistent ~instant re-open timing.
- **Heap reclaim**: Cleared icon cache reclaims the bulk of overlay heap outside Chromium during idle while keeping WebView warm.

### Chores

- **README**: Added Rust badge, fixed stale content, updated warm-release description
- **LICENSE**: Updated copyright to haxllo
- **Config comment**: Updated `ui_warm_release_ms` to reflect icon-cache-only behavior
