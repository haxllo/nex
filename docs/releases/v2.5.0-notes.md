Inter variable font, typography refinements, footer logo support, and pinned list fix.

## Changelog (12 commits since v2.4.3)

### Performance

- **Font**: Switched from single-weight Geist-Regular to InterVariable variable font (100-900 weight range) — real font weights instead of synthetic.

### Bug Fixes

- **Pinned list**: Quick Launch section no longer shows when all items are unpinned — was auto-filling from recent apps.
- **Footer logo aspect ratio**: Fixed stretching by using `height` + `width: auto`.
- **Custom protocol**: Switched from `nexasset://` (silently ignored by WebView2 for `<img>`) to base64 served via Rust `serve_asset`.

### Architecture

- **Typography**: Reduced font sizes by 1px baseline, letter-spacing 0.5px→0.2px, body weight 300→400, placeholder 200→300.
- **Footer logo**: Embedded via `include_bytes!`, base64-encoded at serve time — no inline base64 in HTML.
- **Color tokens**: `--text-dim` changed to `#818181`, placeholder and search icon use `--border-strong` / `--text-dim`.
