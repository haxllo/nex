# Nex v6.4.1 Release Notes

Release date: 2026-05-17

## Summary

v6.4.1 adds proper crates.io publishing and provides official Windows release artifacts.

## What's New

### Publishing
- Published to crates.io as `nex-cli`
- Install via: `cargo install nex-cli`
- Binary name: `nex`

### Release Artifacts
- **nex-6.4.1-windows-x64.zip** - Portable standalone
- **nex-6.4.1-windows-x64-setup.exe** - Windows installer
- **nex-6.4.1-windows-x64-manifest.json** - Checksums and file manifest

## Known Issues

- Game mode detection may not work for all games
- Hotkey conflicts show warning but don't block startup

## Upgrading

Existing users can upgrade by:
1. Running the new installer, or
2. Downloading the portable zip and replacing nex.exe

## Installation

### Via Installer
Download `nex-6.4.1-windows-x64-setup.exe` and run.

### Via Portable
Download `nex-6.4.1-windows-x64.zip`, extract, and run `nex.exe`.

### Via Cargo
```bash
cargo install nex-cli
```
